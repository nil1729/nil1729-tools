# tools.nilanjandeb.com

A collection of browser-based developer tools. No backend, no tracking — everything runs client-side.

Live at **[tools.nilanjandeb.com](https://tools.nilanjandeb.com)**

## Tools

- **JSON Linter & Formatter** — validate, format, and inspect JSON with syntax highlighting

## Stack

- [Next.js 16](https://nextjs.org) (App Router)
- [React 19](https://react.dev)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS v4](https://tailwindcss.com)
- [CodeMirror 6](https://codemirror.net) — editor with JSON language support
- [shadcn/ui](https://ui.shadcn.com) — component primitives
- [Vitest](https://vitest.dev) — unit tests

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
pnpm test       # run tests
pnpm build      # production build
```
